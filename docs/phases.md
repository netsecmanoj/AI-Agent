# Phases

## Phase 1

- Milestone 1: skeleton, health, dashboard, persistence, Semgrep adapter, reports, Docker
- Milestone 2: scan upload/archive ingestion, Trivy integration, dependency audit adapters
- Milestone 3: ecosystem-aware dependency enrichment, history/detail filtering, richer reports, stronger test coverage
- Milestone 4: async queue/worker execution, scan lifecycle states, refresh-friendly UI

## Phase 2

- Milestone 5: basic admin authentication, session-based access control, and project management
- Milestone 17: OIDC/SSO login, user provisioning, and richer RBAC with `admin` / `reviewer` / `viewer`
- Better audit history comparisons
- Search and severity filtering

## Phase 3

- Milestone 6: AI explanation generation, remediation suggestions, and management summaries
- Milestone 10: deterministic duplicate finding grouping and grouped AI/report refinements

## Phase 4

- Milestone 7: GitLab CI template pack, token-authenticated API routes, and webhook-based triggers
- Milestone 8: broader ecosystem coverage with Node dependency auditing and improved ecosystem-aware execution
- Build failure thresholds for critical findings

## Phase 5

- Milestone 9: retention cleanup controls, worker health visibility, and storage safeguards
- Milestone 11: scan-to-scan comparison, regression tracking, and deterministic grouped deltas
- Milestone 12: Flutter/Dart ecosystem detection, static analysis, and dependency freshness coverage
- Milestone 13: Flutter mobile configuration review for Android and iOS posture findings
- Milestone 14: shared ecosystem-adapter refinement plus first-pass Maven and Composer support
- Milestone 15: first-pass Go modules and .NET / NuGet support through the shared adapter registry
- Milestone 16: deterministic CI policy gates and severity-based enforcement summaries
- Milestone 18: deterministic project-level trend summaries across scan history
- Milestone 19: per-project policy presets and override fields on top of global defaults
- Milestone 20: deterministic framework-aware review for Spring, Laravel, Express, and Flutter app structures
- RBAC
- Notifications
- PDF reports
- Deeper SCM and ticketing integrations
